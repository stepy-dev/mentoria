Não sei JavaScript. O programa de mentoria é pra mim?

Não. Entretanto, se você acredita ser capaz de aprender JavaScript até a data de início do programa de mentoria, eu sugiro que você clique aqui para assistir um curso de JavaScript gratuito no YouTube.

A mentoria será individual?

Sim. Você terá 8 sessões individuais de 30 min cada, realizadas pelo Google Meet. Após se tornar aluno, você será convidado a marcar suas sessões de mentoria.

Faremos um projeto prático?

Sim. Você precisará fazer um projeto prático utilizando as tecnologias apresentadas acima, consumindo uma API real. Lembre-se, você não estará sozinho, conte comigo.

Utilizaremos uma back-end real no front-end?

Sim. Preparei para você uma API utilizando PHP com Laravel. Clique aqui caso tenha interesse em ver a documentação da API.

Terei suporte além das 8 sessões pelo Google Meet?

Sim. Além das 8 sessões ao vivo de 30 min cada, você poderá contar com minha ajuda via grupo no WhatsApp ou email.

Qual a duração da mentoria?

O programa de mentoria tem a duração de 1 mês a partir da data de início no dia 04/08/2023. Mais não se preocupe, o conteúdo gravado ficará sempre disponível para você consultar quando precisar.

Terei acesso a um certificado?

Sim. Após entregar o projeto prático e corrigirmos, um certificado será disponibilizado em até 72 horas.

Conseguirei entrar no mercado de trabalho após fazer a mentoria?

Depende. Como mentor, minha missão será desbloquear você para conseguir ir bem no desenvolvimento de aplicações utilizando Vue.js. Não tenho como garantir que conseguirá um trabalho, mais tenho certeza que você aprenderá tudo o que me proponho a te ensinar.
